# CharacterForge

**Русский** 
Генератор персонажей для писателей, художников и игровых мастеров. Выбери пол — и программа создаст случайное имя (в фэнтези-стиле), предысторию и костюм (броня, штаны/юбка, обувь). Подходит для любых сеттингов.

**English**  
Character generator for writers, artists and game masters. Choose gender — and the program will generate a random fantasy-style name, backstory and outfit (armor, legs, footwear). Works for any setting.

---

## Возможности / Features

-  Выбор пола / Gender selection
-  Генерация имени / Name generation
-  Генерация предыстории / Backstory generation
-  Генерация костюма (3 слота) / Outfit generation (3 slots)

---


## Как использовать / How to use

1. Выбери пол персонажа / Choose gender
2. Нажми «Сгенерировать имя» / Click "Generate name"
3. Нажми «Сгенерировать предысторию» / Click "Generate backstory"
4. Нажми «Сгенерировать костюм» / Click "Generate outfit"

Все данные можно обновлять по отдельности в любом порядке.
